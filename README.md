# DOM Masterclass

**A complete, interactive DOM course in a single HTML file — no frameworks, no dependencies, just open and learn.**

---

## What's Inside

14 progressive sections covering everything from tree structure to advanced event patterns, with live demos, sandboxed code editors, quizzes, and a final build challenge throughout.

**Sections:** Introduction → DOM Tree → Node Types → Selecting Elements → Traversal → Content Manipulation → Attributes vs Properties → Styling → Creating Elements → Events → Event Delegation → Advanced Concepts → Practice Lab → Final Challenge

## Features

- **5 code editors** — edit and run JavaScript in-page with live output
- **5 quizzes** — instant feedback with a running score
- **Event flow animator** — visualizes capture → target → bubble phase by phase
- **Live vs static collection demo** — see HTMLCollection vs NodeList diverge in real time
- **XSS demo** — side-by-side textContent vs innerHTML comparison
- **DocumentFragment benchmark** — timed performance comparison
- **Event delegation demo** — live event log showing a single listener handle all items
- **Final challenge** — a card manager app with 6 TODO sections to complete

## Usage

Download `dom-masterclass.html` and open it in any modern browser. No server, build step, or internet connection required.

## Tech

Vanilla HTML, CSS, and JavaScript — no libraries, no frameworks, no external dependencies.
